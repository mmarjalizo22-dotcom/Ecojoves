EcoJoves - README

1) Afegir imatges:
 - Posa les teves fotos dins la carpeta assets/images/.
 - Els noms dels fitxers han de coincidir amb els que apareixen al codi (ex.: argan.jpg, rosa-mosqueta.jpg).
 - Si canvies nom de fitxer, edita la ruta de l'atribut src a les pàgines HTML.

2) Com editar textos:
 - Obre qualsevol .html amb el Bloc de notes (o Visual Studio Code).
 - Canvia títols, descripcions i preus. Desa els canvis.

3) Com fer el zip a Windows:
 - Selecciona la carpeta EcoJoves, clic dret → "Enviar a" → "Carpeta comprimida (zip)".
 - Això generarà EcoJoves.zip.

4) Pujar a GitHub Pages:
 - Crea un repositori nou a GitHub (nom públic).
 - Puja el contingut de la carpeta EcoJoves al repositori (puets arrossegar i fer "Upload files").
 - A GitHub, ves a Settings → Pages → Source → Branch: main → / (root) → Save.
 - Espereu uns minuts; l'URL serà https://<el-teu-usuari>.github.io/<nom-del-repositori>/

5) Com provar localment:
 - Fes doble clic a index.html per obrir la pàgina al navegador.
 - Navega entre categories per comprovar enllaços.

6) Notes:
 - Els botons "Comprar" són només de mostra (no connectats).
 - Si vols que el botó obri un correu, canvia l'enllaç a: mailto:teucorreu@exemple.com?subject=Comanda%20Oli%20d'Argan
